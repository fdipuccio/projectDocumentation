### 1.2 Architettura Tecnica

#### 1.2.1 Frontend

- **Framework:** Angular (versione 20)
- **Linguaggio:** TypeScript
- **Routing:** Angular Router
- **Form:** Reactive Forms
- **Comunicazione HTTP:** HttpClient (REST, JSON)

#### 1.2.2 Backend

- **Linguaggio:** Java 17+
- **Framework:** Spring Boot
- **Moduli principali:**
  - Spring Web (REST Controller)
  - Spring Data JPA
  - Spring Security
- **Autenticazione:** JWT (token-based)

#### 1.2.3 Database

- **Tipo:** Database relazionale  
- **Esempi:** MySQL, PostgreSQL

#### 1.2.4 Comunicazione

- **Pattern:** REST API (JSON)
- **Protocollo:** HTTP/HTTPS
- **Architettura:** stateless con autenticazione basata su token (JWT)
